Aplicaes

Essas tcnicas tm aplicaes abrangentes para a resoluo de problemas reais, alguns exemplos so:

  • Detalhamento de imagens como pr-processamento para modelos de Machine Learning, a fim de aumentar o desempenho;
  • Melhoramento de imagens de cmeras de segurana ou policiais, a fim de facilitar julgamentos ou identificao de suspeitos;
  • Reconstruo de imagens antigas ou de cmeras mais limitadas;
  • Qualquer outro uso que se beneficia de uma imagem melhor e mais limpa.

O Dataset: RELLISUR (Real Low-Light Image Super-Resolution)

Paper: Andreas Aakerberg, Kamal Nasrollahi, and Thomas B. Moeslund. RELLISUR: A Real Low-Light Image Super-Resolution Dataset. Version 1.0.0. Zenodo, August 23, 2021. https://doi.org/10.5281/ZENODO.5234969.

  • Objetivo: integrar problemas de melhoramento de imagem por Super Resolution, ou por Low Light Enhancement.
  • No geram imagens de baixa resoluo e luminosidade digitalmente, mas fazem as diferenciaes j na captura das imagens
    • Resoluo: diferentes valores de zoom, alterando lentes com diferentes distncias focais;
    • Iliminao: diferentes valores de exposio;

Resoluo

As imagens de para Super Resolution so divididas em 3 tipos:

(menor qualidade) 1x -> 2x -> 4x (maior qualidade)

Baixa Luminosidade

As imagens para Low Light Enhancement so divididas em 5 tipos:

(Mais clara) 2.5 -> 3.0 -> 3.5 -> 4.0 -> 4.5 (Mais escura)

Utilizao do Dataset

Objetivo:
Preencher lacuna de melhoramento completo: Resoluo + Iluminao em treinamento end-to-end

Estrutura do Dataset

Manipulaes aplicadas:

  • Iluminao

Caractersticas dos Dados

Iluminao

Todas as imagens: 625x625 pixels (1x)
Problema: Imagens escuras demais podem dificultar validao visual (ex: pssaros, artefatos)

Resoluo

  • 1x = 625 x 625
  • 2x = 1250 x 1250
  • 4x = 2500 x 2500

Problemas:

  • Capturas rpidas podem ter pequenas diferenas entre frames
  • Recortes podem ser irregulares (apesar de SURF features + MSAC)

Contexto das Imagens

Distribuio balanceada:

  • Ambiente externo: ~51%
  • Ambiente interno: ~49%

Evita modelo enviesado

Privacidade

Dataset sem informaes identificveis:

  • Sem pessoas/rostos
  • Sem placas de carro
  • Sem contedo ofensivo

Limitao: Modelo pode performar pior em imagens com essas caractersticas/

Planejamento Inicial

Pr-Processamento

Data Augmentation:

  • Rotao e translao de imagens
  • Aplicar mesmas transformaes s imagens ground-truth

Vantagem do dataset: No possui desbalanceamento ou problemas comuns pela natureza do problema

Experimentos Iniciais

Objetivo: Comear pelo mais simples para validar mtodo de treinamento

Arquitetura:

  • CNNs convencionais
  • Otimizador: ADAM

Loss Functions:

  • MSE ou PSNR

Estratgia:

  • Separar problemas: treinar modelos isolados
    • Super Resolution apenas
    • Low Light Enhancement apenas

Expectativa: Resultados modestos, mas suficientes para validao do mtodo

Prximos Passos

  • Testar outras arquiteturas de redes: U-Net, Res-Net, Redes Adversrias;
  • Testar outras losses: procurar outras mtricas por pxel, e tambm por percepo;
  • Juntar o problema: treinamento end-to-end;
